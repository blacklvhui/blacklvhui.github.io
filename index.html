ubuntu 14.04下怎样交换Caps和ctrl的位置

首先，安装xmodmap

sudo apt-get install xmodmap

如果提示没有则安装

sudo apt-get install x11-xserver-utils

在Home中建立文件.Xmodmap

gedit .Xmodmap

然后输入

remove Lock = Caps_Lock  
remove control = Control_L  
keysym Control_L = Caps_Lock  
keysym Caps_Lock = Control_L  
add lock = Caps_Lock  
add control = Control_L

最后运行

xmodmap .Xmodmap

关于指针的用法（1）

一个类就是一个用户定义的类型。struct也是一个类，不过struct是值类型保存的是具体的值，而类是引用型，保存的是值的地址。

类的一般使用方法：

class 类名
{
public://class的public成员，任何可以看到这个class的地方都可以访问这些成员。   
    ...
private://class的private成员，只有同一个class的其他成员或该class的“friend” class可以访问这些成员。
    ...
};

类的引用：

#include "User.h"

auto u = new User;//实例化，u为Uers类型的指针，把实例化的Uers的地址赋给u
u->InputName(str);//调用了User的成员函数InputName(std::string str)

类中的函数成为成员函数。

class Table
{
    Name *p;
    size_t sz;
public:
    Table(size_t s=15){p = new Name[sz=s];} //构造函数，构造函数与类名相同,主要用来在创建对象时初始化对象， 即为对象成员变量赋初始值，总与new运算符一起使用在创建对象的语句中。
    ～Table() {delete[] p;} //析构函数,完成一些清理和释放的工作
}

指针的用法

指针的一般定义方法：

int a = 1;
int *p = &a;

p为a的地址
*p就是a

假设a被放在内存中地址1000的单元中，那么p的值为1000，*p的值为1。

#include<iostream>

int main()
{
    char a[2]={'a','b'};

    char *p = &a[0];




        std::cout << ++*p <<std::endl;
return 0;
}

数组分配的是一段连续的内存
如果a[0]存的是a，假设a的地址为1000，在32位的系统中，char为4位，那么b的地址就是10004。
如果指针p指向数组a[]的首地址也就是a,那么++p就是指向数组a[]的首地址的下一个地址也就是b,所以输出的++*p就是b。

static 的使用方法

blacklvhui
static是静态变量的一种用法，例如：static int n;
用了static，在函数内部时，另一个函数对static所在函数进行调用，当static所在函数执行过后，static定义的变量值不会返回初值，他会发生改变并且不会释放内存。
例如：

#include<iostream>

int f()
{
    static int a=0; // 初始化a，使用static关键字后，只在第一次调用的时候初始化a，并且a只会被初始化一次
    a++;　//　a = 1

    return a; // return 1
}

int fn()
{
    int a = 0;　// 每次调用该函数时都会初始化a
    a++;    // a = 1

    return a; // return 1
}

int main()
{
    // 调用用static修饰变量的函数f()
    std::cout << f() << std::endl; // 输出1
    std::cout << f() << std::endl; // 输出2
    std::cout << f() << std::endl; // 输出3
    std::cout << f() << std::endl; // 输出4

    // 调用没有用static修饰变量的函数f()
    std::cout << fn() << std::endl; // 输出1
    std::cout << fn() << std::endl; // 输出1
    std::cout << fn() << std::endl; // 输出1
    std::cout << fn() << std::endl; // 输出1
    return 0;
}
