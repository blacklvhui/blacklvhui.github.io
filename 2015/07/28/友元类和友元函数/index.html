友元类和友元函数

类可以允许其他类或者函数访问它的非公有成员，方法是令其他类或者函数成为它的友元。

友元类

class Merchant
{
private :
    int m_MyMoney;
    int m_MyRoom;
    … …
public:
    Friend class Lawyer; //友元类的声明
    Int getmoney();
    … …
};

class Lawyer //赋予Lawyer类为Merchant为友元，Lawyer类就有了访问Merchant类的私有数据
{
private:
    … …
public:
    … …
};

友元函数

class Merchant
{
private :
    int m_MyMoney;
    int m_MyRoom;
    … …
public:
    Friend void Lawyer(Merchant &obj); //友元函数的声明
    Int getmoney();
    … …
};

void Lawyer(Merchant &obj) //赋予函数Lawyer为Merchant为友元，函数Lawyer就有了访问Merchant类的私有数据
//不能写成Merchant::void Lawyer(Merchant &obj)
{
    std::cout << obj.m_MyMonry << std::endl;    
}

一个普通函数可以是多个类的友元函数

class Merchant
{
public:
    Friend void Lawyer(Merchant &obj); //友元函数的声明
    Int getmoney();
    … …
private :
    int m_MyMoney;
    int m_MyRoom;
    … …
};

class Country
{
public:
    Friend void Lawyer(Merchant &obj); //友元函数的声明
    … …
private:
    int m_MyMoney;
    … …
};

void Lawyer(Merchant &obj) //赋予函数Lawyer为Merchant为友元，函数Lawyer就有了访问Merchant类的私有数据
//不能写成Merchant::void Lawyer(Merchant &obj)
{
    std::cout << obj.m_MyMonry << std::endl;    
}

一个类的成员函数函数也可以是另一个类的友元，从而可以使得一个类的成员函数可以操作另一个类的数据成员

class Merchant
{
public:
    void Lawyer(Merchant &obj) //成员函数的声明
    Int getmoney();
    … …
private :
    int m_MyMoney;
    int m_MyRoom;
    … …
};

class Country
{
public:
    Friend void Merchant::Lawyer(Merchant &obj); //友元函数的声明
    … …
private:
    int m_MyMoney;
    … …
};
