C++继承

通过继承机制，可以利用已有的数据类型来定义新的数据类型。所定义的新的数据类型不仅拥有新定义的成员，而且还同时拥有旧的成员。
A类继承B类，我们称B类为“基类”，也称为“父类”，A类则相应地被称为“派生类”或“子类”。A类继承了B类之后，A类就具有了B类的部分成员。

单继承的定义格式

class<派生类名>:<继承方式><基类名>
{
<派生类新定义成员>
};

有如下三种继承方式：
1.公有继承（public）
2.保护继承（protected）
3.私有继承（private）

在派生类内部：
                public       protected       private
    共有继承     public          protected          不可见
    私有继承    private          private          不可见
    保护继承    protected     protected          不可见

在派生类外部：
                public          protected          private
    共有继承     public            不可见               不可见
    私有继承     不可见          不可见            不可见
    保护继承     不可见          不可见            不可见

继承示例（公有继承）：

在Master.h中

class Master
{

#pragma mark - Init
public:
    Master();
    ~Master();

#pragma mark - Get&Set
public:
    void OutPutPublic();

protected:
    void OutPutProtected();

#pragma mark - Variable
private:
    void OutPutPrivate();
};

在Master.cpp中

#include "Master.h"
#include<iostream>

Master::Master()
{
}

void Master::OutPutPublic()
{
    std::cout << "This is a Public Method" << std::endl;
}

void Master::OutPutProtected()
{
    std::cout << "This is a Protected Method" << std::endl;
}

void Master::OutPutPrivate()
{
    std::cout << "This is a Private Method" << std::endl;
}

若类A要用公有继承方式继承类Master

在A.h中
class A : public Master
{
public:
void PublicMethod();
};

在A.cpp中

void A::PublicMethod()
{
    OutPutPublic();  //由于是公有继承，所以子类A可以访问基类Master的公有成员和保护成员
    OutPutProtected();
}
