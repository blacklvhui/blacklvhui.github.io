关于类和指针的用法（1）

一个类就是一个用户定义的类型。struct也是一个类，不过struct是值类型保存的是具体的值，而类是引用型，保存的是值的地址。

类的一般使用方法：

class 类名
{
public://class的public成员，任何可以看到这个class的地方都可以访问这些成员。   
    ...
private://class的private成员，只有同一个class的其他成员或该class的“friend” class可以访问这些成员。
    ...
};

类的引用：

#include "User.h"

auto u = new User;//实例化，u为Uers类型的指针，把实例化的Uers的地址赋给u
u->InputName(str);//调用了User的成员函数InputName(std::string str)

类中的函数成为成员函数。

class Table
{
    Name *p;
    size_t sz;
public:
    Table(size_t s=15){p = new Name[sz=s];} //构造函数，构造函数与类名相同,主要用来在创建对象时初始化对象， 即为对象成员变量赋初始值，总与new运算符一起使用在创建对象的语句中。
    ～Table() {delete[] p;} //析构函数,完成一些清理和释放的工作
}

list的用法如下：

#include <iostream>
#include <list>

int main()
{
    std::list<User*> users;

return 0;
}

压到链表的最后一个：

users.push_back(u);

强制转换语法如下：

auto str = (std::string)"0";

for循环的一种用法如下：

for(auto p : users)
    {
        i++;
        if(3 > i) continue;
        p->PrintNamePrice();
    }
